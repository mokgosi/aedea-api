# 🚀 Symfony API Backend

This is the backend API for idea submissions, collaborations, project proposals or business enquires built with:

- Symfony 8
- API Platform
- Doctrine ORM
- JWT Authentication
- reCAPTCHA verification
- Symfony Mailer 
- Mailpit 

---

# ⚙️ Requirements

- PHP 8.1+
- Composer
- Symfony CLI (optional but recommended)
- MySQL or PostgreSQL
- Node (optional for frontend)

---

# 📥 Installation

## 1. Clone main app

```bash
$ git clone git@github.com:mokgosi/aedea-app.git
$ cd aedea-app
```

## 2. Clone api

```bash
$ git clone git@github.com:mokgosi/aedea-api.git

```

## 3. Clone frontend

```bash
$ git clone git@github.com:mokgosi/aedea-api.git

```

## 4. Execute

```bash
$ ./scripts/setup.sh

```
